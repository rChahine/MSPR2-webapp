# Stucture du projet

```
├── build  # Fichiers de build de l'application
|
├── Config  # Fichiers d'environnement
|
├── static  # Assets de l'application
|
├── tests/unit  # Contient les fichiers de tests de l'application
|
├── src
│   ├── components  # Composants vuejs
│   ├── assets  # Assets des composants
|   ├── router  # Liste des routes de l'application
|   └── App.vue  # Composant mère de l'application
|   └── main.js  # Point d'entrée de l'application
│       
└── .gitignore
└── composer.json # Fichier de dépendances
```
